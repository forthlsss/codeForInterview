#### 题目
输入两个链表，找出它们的第一个公共结点。
#### 思路
因为当出现一个公共节点时，后面的节点都相同（因为链表的后继节点是唯一的，也就表明公共节点的后继一定相同，以此类推，后面的节点都相等），因此可以将所有节点放入栈中，从后往前看，直到不相同的点，那么他的前一个节点就是第一个公共节点了。
